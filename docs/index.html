Quick Start

1. Install Dependencies

pip install markdown Pygments

2. Basic Usage

# Convert Markdown to HTML presentation
python slidown.py input.md output.html

# Choose a theme
python slidown.py input.md output.html --theme clean

# Custom split level (H1 headings)
python slidown.py input.md output.html --split-level 1

# Add custom footer
python slidown.py input.md output.html --footer "¬© 2026 Your Company"
