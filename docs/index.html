Quick Start

1. Install Dependencies

pip install markdown pygments

2. Clone and Convert

# Clone the repository
git clone https://github.com/dwHou/slidown.git
cd Slidown

# Convert Markdown to HTML presentation (auto-generated timestamped folder)
python slidown.py your_notes.md

# Choose a theme
python slidown.py your_notes.md --theme clean

# Add custom footer
python slidown.py your_notes.md --footer "¬© 2026 Your Company"

Usage Examples

1. Basic Conversion
python slidown.py lecture.md
2. Specify Theme and Output
python slidown.py notes.md -o ~/Desktop/presentation --theme clean
3. Corporate Presentation (with custom footer)
python slidown.py pitch.md --theme corporate --footer "Confidential - ¬© 2026 Company Inc."
4. Adjust Pagination and Navigation
python slidown.py doc.md --split-level 1 --chapter-level 3 --viewport-height 1080
5. Preserve Image Paths (no copying)
python slidown.py article.md --preserve-image-paths
6. Smart Pagination (custom content length)
python slidown.py long_tutorial.md --max-content-length 600 --max-elements 12

Advanced Features

LaTeX Math Formulas

Full support for mathematical notation using KaTeX.

# Inline math
The equation $E=mc^2$ is Einstein's mass-energy equivalence.

# Block math
$$
\int_{-\infty}^{\infty} e^{-x^2} dx = \sqrt{\pi}
$$

Smart Pagination

Automatically splits long content while preserving integrity.

  • Based on viewport height and content threshold
  • Protects code blocks, tables, and formulas from splitting
  • Configurable via --viewport-height and --content-threshold
  • Ensures no scrolling required on each page

Dual Navigation System

Two complementary navigation methods for better orientation.

  • Chapter Progress Bar (bottom) - Shows major chapters (H1-H2 by default), current position highlighted
  • TOC Panel (top-left) - Full document outline (H1-H5), expandable/collapsible
  • Configure chapter level with --chapter-level (1-6)
  • Disable with --no-chapter-nav if needed

Flexible Image Handling

Choose between two image processing modes.

Default Mode (Copy)
  • Images copied to assets/images/
  • Self-contained presentation
  • Easy sharing and archiving
Preserve Mode
  • Keep original image paths
  • No file copying
  • HTML in same folder as Markdown
